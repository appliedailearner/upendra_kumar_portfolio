## ğŸ“‹ Quick Start (5 Steps)

### Step 1: Create Portfolio Repository (2 minutes)

1. Go to GitHub: https://github.com/new
2. Repository name: `[your-username].github.io`
   - **Example**: If your username is `appliedailearner`, name it `appliedailearner.github.io`
3. Set to **Public**
4. Check "Add a README file"
5. Click "Create repository"

### Step 2: Enable GitHub Pages (1 minute)

1. Go to repository Settings
2. Scroll to "Pages" section (left sidebar)
3. Under "Source", select:
   - Branch: `main`
   - Folder: `/ (root)`
4. Click "Save"
5. Wait 1-2 minutes for deployment

**Your site will be live at**: `https://portfolio.upendrakumar.com/`

### Step 3: Clone Repository Locally (1 minute)

```powershell
cd c:\MyResumePortfolio
git clone https://github.com/[your-username]/[your-username].github.io.git
cd [your-username].github.io
```

### Step 4: Add Portfolio Files (5 minutes)

I'll create the files for you - just copy them to your repository!

### Step 5: Push to GitHub (1 minute)

```powershell
git add .
git commit -m "Initial portfolio website"
git push origin main
```

**Done!** Your portfolio will be live in 1-2 minutes.

## ğŸ†˜ Troubleshooting

**Site not loading?**
- Wait 2-3 minutes after first push
- Check GitHub Pages settings
- Ensure repository is public
- Verify branch is set to `main`

**Changes not showing?**
- Clear browser cache (Ctrl+F5)
- Wait 1-2 minutes for rebuild
- Check commit was successful

**404 Error?**
- Verify file names are correct
- Check file extensions (.html)
- Ensure files are in root directory
