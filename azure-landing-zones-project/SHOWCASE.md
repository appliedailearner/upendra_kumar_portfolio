### When Asked: "How do you ensure security?"

"I implement defense-in-depth with multiple layers:
- Network layer: Azure Firewall, NSGs, private endpoints only
- Identity layer: Azure AD with MFA and conditional access
- Data layer: Customer-managed encryption keys, TLS 1.3
- Monitoring layer: Azure Sentinel for threat detection

Everything is validated through automated security scanning with Checkov, TFSec, and Trivy - over 100 security checks on every deployment. I also use Azure Policy-as-code to enforce compliance continuously, not just at deployment time."

### When Asked: "How do you manage costs?"

"I take a FinOps approach with four strategies:
1. Reserved Instances for predictable workloads - 40% savings
2. Autoscaling with scale-to-zero for variable workloads
3. Storage lifecycle management to move data to cheaper tiers
4. Spot instances for non-critical training jobs

I also implement budget alerts with automated actions and use Infracost in the CI/CD pipeline so teams know the cost impact before deploying. This delivered $55,000 in annual savings on this project."
